Project Overview
This is a fully functional Object-Oriented Programming (OOP) based accounting system for real-life personal financial management. It supports income and expense tracking, balance calculation, category and monthly statistics, and data persistence via JSON file. The project fully implements all required OOP concepts and provides a user-friendly console interface.
Core Features
Add and manage income / expense records
Automatic total income, total expense, and balance calculation
Category-based statistics and monthly filtering
Data validation for date and amount
Save and load records using JSON file
Clear console menu and error handling
OOP Concepts Implemented
Class & Object
Constructor (__init__)
Encapsulation (private attributes + getter methods)
Inheritance (base class + subclasses)
Polymorphism (overridden methods)
Abstraction
